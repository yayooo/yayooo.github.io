针对windows用户
需要hexo，node.js，git

下载安装git，之后会出一篇自己的总结，可以参考廖雪峰的教程
下载安装node.js，选最新的版本就行，参考这两个教程
安装hexo

注意的是

$ hexo s



之后会出现

$ hexo s

INFO Start processing

INFO Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.



且后续未出现任何东西，表明现在可以本地预览，ctrl+c之后便退出预览功能。

git 的复制粘贴建议用右键后Copy，Paste使用

配置文件修改中每个：后需要一个空格否则会在

$ hexo g

$ hexo s



后报错
