下载安装git

参考廖雪峰的教程
*git 的复制粘贴建议用右键后Copy，Paste使用

下载安装node.js

选最新的版本就行，参考菜鸟教程
*安装模式注意选 Add to PATH

*在命令提示符中输入node -v后提示不是内部或外部命令，也不是可运行的程序时需要重新配置环境变量。

我的电脑
属性
高级系统设置
环境变量
path
新建一个地址为node.js安装地址,如我的是D:/fe/node.js

安装hexo

参考使用Hexo搭建个人博客(基于hexo3.0)
*$ hexo s后出现下面所示且后续无任何东西，表明现在可以本地预览，ctrl+c之后便退出预览功能。

$ hexo s
INFO  Start processing
INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

*配置文件修改中每个：后需要一个空格否则会在

$ hexo g
$ hexo s

后报错
